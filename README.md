# WolfsNet — Site estático

Site estático demonstrativo para a provedora de internet WolfsNet. Projeto criado para UI/UX minimalista, com tema escuro e acentos em ciano.

Arquivos:
- `index.html` — página principal
- `styles.css` — estilos e responsividade
- `script.js` — animação de partículas e interações leves
- `logo.svg` — logotipo em SVG

Como abrir:
1. Abra `index.html` no seu navegador (duplo clique ou "Abrir com")

Notas de design:
- Paleta: tons de cinza, ciano e preto; tipografia Montserrat/Orbitron
- O logo é um SVG simples e pode ser substituído por uma versão profissional.
- Layout responsivo: desktop, tablet e celular.
